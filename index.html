HTML

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for Web pages. HTML elements are the building blocks of HTML pages. Some resources for learning HTML are:

  • Progate
  • W3Schools
  • FreeCodeCamp

CSS

CSS is a language that describes the style of an HTML document. CSS describes how HTML elements should be displayed. Some resources for learning CSS are:

  • Progate
  • FreeCodeCamp
  • Udemy

JS

Javascript is the programming language of HTML and the Web. It is easy to learn. It can calculate, manipulate and validate data. It can update and change both HTML and CSS. Some resources for learning JS are:

  • Progate
  • Udemy
  • MDN

REACT

React is a Javascript library created by Facebook. It is a User Interface library. It is a tool for building UI components. Some resources for learning React are:

  • Udemy
  • Progate
  • FreeCodeCamp
